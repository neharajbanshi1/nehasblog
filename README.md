# *ੈ✩‧₊˚ Neha's Data Projects

A personal blog showcasing data analysis projects, built with **Astro 6** + **Tailwind CSS v4**.

✦ **Live site**: https://nehasblog.vercel.app
✦ **Stack**: Astro, Tailwind CSS, MDX, KaTeX
✦ **Deploy**: Vercel

## ✧˖° Quick Start

```bash
npm install
npm run dev     # local dev at http://localhost:4321
npm run build   # build for production
npm run preview # preview production build
```

## 📂 Project Structure

```
src/
├── components/     # Astro components
├── content/
│   └── blog/       # Blog posts (Markdown/MDX)
├── layouts/        # Page layouts
├── pages/          # Route pages
└── styles/         # Global CSS
```

## 🏷️ Adding a Post

Create a `.md` or `.mdx` file in `src/content/blog/`:

```yaml
---
title: "Your Project Title"
description: "Brief description"
date: 2026-06-15
tags: [python, pandas, visualization]
category: Projects
cover: ""
pinned: false
---
```
